# /manager role

Add or remove a role as a manager. All members with the role can configure AutoReacter.

## Permissions

- **Requires:** Administrator

## Usage

Fill in the parameter below and run `/manager role`.

## Parameters

### 1. **Role** (Required)
- The role to add or remove as a manager
- Running the command again with the same role removes it
- All members with the role can manage auto-reactions
