# /manager member

Add or remove a specific member as a manager. The member can configure AutoReacter without admin permissions.

## Permissions

- **Requires:** Administrator

## Usage

Fill in the parameter below and run `/manager member`.

## Parameters

### 1. **Member** (Required)
- The member to add or remove as a manager
- Running the command again with the same member removes them
- The member can manage auto-reactions without administrator permissions
