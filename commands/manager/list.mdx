# /manager list

View all roles and members currently assigned as managers in your server.

## Permissions

- **Requires:** Administrator or Manager

## Usage

`/manager list`

## What It Shows

- **Manager roles** - All roles with manager permissions
- **Manager members** - All individual members with manager permissions
- **Count** - How many of each you have (ex: 2/5 members used)
