# Manager

Managers are users and roles you trust to configure AutoReacter without needing administrator permissions.

## What Can Managers Do?

Managers can:
- Add and remove auto-reactions
- Modify auto-reaction settings
- Manage whitelist and blacklist
- View auto-reaction list
- View manager list

Managers **cannot**:
- Add/remove other managers
- Change server settings

## Limits

| Item | Free | Premium |
|---|---|---|
| Manager roles | 2 | 30 |
| Manager members | 5 | 50 |

## Commands

- `/manager role` - Add/remove manager roles
- `/manager member` - Add/remove manager members
- `/manager list` - View all managers
