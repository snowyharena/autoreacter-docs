# /premium deactivate

Remove premium features from your server and free up a premium slot.

## Usage

`/premium deactivate`

## What It Does

- Disables premium in the server
- Frees up your premium slot for another server
- Server returns to free tier limits (10 auto-reactions, lower limits)
