# AutoReacter+

AutoReacter Premium unlocks significantly higher limits and faster reaction speeds for **$3.99/month.**

### What's Included?

| Feature | Free | Premium |
|---|---|---|
| **Auto-reaction slots** | 10 | **500** |
| **Reaction speed** | Standard | **Faster** |
| **Priority support** | <Icon icon="x" size={12} /> | **<Icon icon="check" size={12} />** |
| Whitelist roles | 2 | **30** |
| Whitelist members | 5 | **50** |
| Whitelist words | 2 | **30** |
| Whitelist word character limit | 30 | **200** |
| Blacklist roles | 2 | **30** |
| Blacklist members | 5 | **50** |
| Blacklist words | 2 | **30** |
| Blacklist word character limit | 30 | **200** |
| Manager roles | 2 | **30** |
| Manager members | 5 | **50** |

## Commands

- `/premium activate` - Activate premium in your server
- `/premium deactivate` - Deactivate premium in your server
- `/premium status` - Check if your server has premium active
- `/premium info` - View premium features and pricing
