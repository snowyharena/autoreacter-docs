# /ping

Check the bot's response time and system information.

## Usage

Fill in the optional parameter below and run `/ping`.

## Parameters

### 1. **Ephemeral** (Optional)
- Whether the response is only visible to you
- Default: True

## What It Shows

- **Bot latency** - Response time in milliseconds
- **Shard latency** - Current shard response time
- **Shard ID** - Current shard number
