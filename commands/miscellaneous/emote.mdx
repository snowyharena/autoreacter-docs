# /emote

Get enlarged emoji images and links for use in embeds.

## Usage

Fill in the parameter below and run `/emote`.

## Parameters

### 1. **Emoji** (Required)
- The emoji(s) to view
- Separate multiple emojis with spaces
- Up to 100 emojis at once
- Supports Unicode emojis (😀, 🎉, 🌟), custom server emojis, and animated custom emojis

## What It Shows

For each emoji:
- **Enlarged emoji image** as file attachment
- **Emoji name**
- **Link for emoji URLs** (clickable in embed field)

## Pagination

Shows 10 emojis per page with navigation buttons for larger batches.
