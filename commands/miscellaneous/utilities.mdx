# Miscellaneous

Miscellaneous commands are utility and quality-of-life commands for AutoReacter.

## Commands

- `/clone` - Clone emojis from other servers
- `/emote` - View enlarged emoji details
- `/ping` - Check bot latency and system info
- `/help` - View command help menu
- `/support` - Get support server link
- `/invite` - Get bot invite link
- **Get Emojis** - Context menu to extract emojis from messages
