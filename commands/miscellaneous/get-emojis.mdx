# Get Emojis

Extract and view emoji images and links from any Discord message using the context menu.

## Usage

1. Right-click any Discord message
2. Select **Apps** from the menu
3. Find and Hover **AutoReacter**
3. Click **Get Emojis**
4. The bot displays all emojis from that message

## What It Shows

For each emoji in the message:
- **Enlarged emoji image** as file attachment
- **Emoji name**
- **Link for emoji URLs** (clickable in embed field)

## Pagination

Shows 10 emojis per page with navigation buttons for messages with many emojis.
