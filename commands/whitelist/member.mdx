# /whitelist member

Add or remove a specific member from the whitelist of an auto-reaction. The bot will only react to messages from whitelisted members.

## Permissions

- **Requires:** Administrator or Manager

## Usage

Fill in the parameter below and run `/whitelist member`.

1. Select whether to add or remove the member
2. Select the auto-reactions to apply it to and confirm

## Parameters

### 1. **Member** (Required)
- The member to add or remove from the whitelist

## Pagination

If you have more than 20 auto-reactions:
- Results show 20 per page
- Use the back and forward buttons to navigate pages
- Switching pages resets your current selection
