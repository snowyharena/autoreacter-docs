# /autoreaction reorder

Change the priority order of your auto-reactions within a channel. When a channel has multiple auto-reactions, you can decide which reaction triggers first.

## Permissions

- **Requires:** Administrator or Manager

## Usage

`/autoreaction reorder`

1. Select the auto-reaction you want to reorder from the dropdown and confirm
2. Choose the new order via the numbered buttons

## Important Notes

<Note>
Only channels with 2 or more auto-reactions can be reordered. If a channel has only one auto-reaction, it won't appear in the dropdown.
</Note>

## Pagination

If you have many reorderable channels:
- Results show 20 per page
- Use the back and forward buttons to navigate pages
- Select an auto-reaction from any page, order buttons will appear after confirmation
