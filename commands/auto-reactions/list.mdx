# /autoreaction list

View all auto-reactions configured in your server with details about each one.

## Permissions

- **Requires:** Administrator or Manager

## Usage

`/autoreaction list`

## What It Shows

For each auto-reaction:
- **Emoji** - The reaction emoji
- **Channel** - Where the reaction is active
- **Type** - Message filter type (All Messages, Text Only, etc.)
- **Modifications** - Any settings applied (ignore bots, delete unreacted, etc.)
- **Whitelist/Blacklist** - Applied filters

## Pagination

If you have many auto-reactions:
- Results show 3 per page
- Use the back and forward buttons to navigate pages
- Total count displayed at top
