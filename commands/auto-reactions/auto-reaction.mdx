# Auto-reactions

AutoReacter's core feature: automatically react to messages in your Discord server with customizable emoji reactions.

## What Are Auto-reactions?

Auto-reactions automatically add emoji reactions to messages in a channel based on filters and conditions you set. When a message matches your criteria, the bot automatically reacts with your chosen emoji.

## Key Features

### 1. **Message Type Filtering**
Choose which types of messages to react to:
- All Messages
- Text Only
- Image Only
- Video Only
- Media Only (images + videos)
- Link Only

### 3. **Whitelist & Blacklist**
Control who can trigger reactions:
- Whitelist specific roles, members, or words to allow reactions
- Blacklist specific roles, members, or words to deny reactions

## Commands

- `/autoreaction add` - Add an auto-reaction to a channel
- `/autoreaction list` - View all auto-reactions in your server
- `/autoreaction modifications` - Adjust settings for an auto-reaction
- `/autoreaction remove` - Remove one or more auto-reactions
- `/autoreaction reorder` - Change the order of auto-reactions

## Use Cases

- **Fun reactions** - Auto-react with fun emojis to create engagement
- **Automation** - Reduce manual reaction work for high-volume channels
- **Ranking systems** - Set up number reactions (1️⃣ through 🔟) so members can rate posts on a scale

And much more. auto-reactions can be applied to almost any use case in your server.
